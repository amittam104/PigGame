Game Rules

  • The game can be played with 2 players.
  • First of all add both the player names.
  • Then provide the maximum number to be achieved to win the game. Then Start the game.
  • In each turn, a player rolls a dice as many times as he/she wants. Each result gets added to the CURRENT score.
  • But, if the player rolls a 1, the CURRENT score becomes 0. After that, it's the next player's turn.
  • The player can choose to 'Hold', which means that the CURRENT score gets added to the TOTAL score. After that, it's the next player's turn.
  • The first player to reach MAXIMUM points on GLOBAL score wins the game.
